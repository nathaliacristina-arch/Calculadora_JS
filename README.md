# Calculadora_JS
JavaScript (JS) é uma linguagem de programação interpretada, dinâmica e multiparadigma, amplamente utilizada no desenvolvimento web para criação de interfaces interativas.

Neste projeto, foi desenvolvida uma calculadora utilizando JavaScript puro, com foco na aplicação de conceitos como manipulação do DOM, tratamento de eventos, funções, estruturas condicionais e controle de estado da aplicação. A lógica implementada permite a realização de operações matemáticas básicas (adição, subtração, multiplicação e divisão).
